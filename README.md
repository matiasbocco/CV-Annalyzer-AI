# CV-Annalyzer-AI

AI tool that analyzes resume–job description compatibility using LLMs.
Returns a compatibility score, strengths, skill gaps, and actionable recommendations.

Stack
- **Core API:** Python · FastAPI · OpenAI GPT-4o mini
- **Channel:** WhatsApp Business API
- **DB:** MySQL · SQLAlchemy
- **Vector DB:** ChromaDB (Phase 3)
